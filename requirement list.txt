requirement/use case

사용자는 의류 쇼핑 사이트 사용 권한 얻기 위해 회원가입 해야한다/사용자 추가, 사용권한발생 event 지정
회원가입은 자신의 기본 정보(이름, 주민번호, 주소, 이메일)과 ID/Password를 저장해야한다/회원가입 추가

회원 탈퇴 기능을 만들어야 하며, 탈퇴와 동시에 시스템 사용 권한 소멸된다/회원추가, 탈퇴 지정
탈퇴와 동시에 시스템 사용 권한 소멸된다/시스템 사용권한소멸 event를 탈퇴에 연결한다

회원은 자신이 판매 완료한 의류 상품 조회할 수 있다, 상품명의 오름차순으로 정렬해서 출력하며
각 상품에 대해 상품명, 제작회사명, 가격, 판매된 수량, 평균 구매만족도를 출력한다/판매 완료의류상품조회 추가
판매 완료한 상품은 판매 종료일이 지났거나 남은 수량이 0이 된 상품을 의미한다/판매 종료일 지난 event, 남은 수량 0 event추가 후 판매완료에 지정
검색 기능을 검색 조건(상품명, 평균 구매만족도)에 맞춰 입력하면 상품 리스트 출력한다/검색 추가
구매 원하는 상품 선택하면 해당 상품 상세한 정보 출력된다/구매 원하는 상품 선택 추가한다
구매 기능이 구현되야 한다/구매 추가
