회원 가입
1.사용자가 기본 정보로 이름, 주민번호, 주소, 이메일을 입력하고 ID/Password를 입력한다.  //2. 회원 가입 완료 메시지를 띄운다
									    
탈퇴
1.회원이 탈퇴 버튼을 입력한다.	//2.탈퇴와 동시에 시스템 사용 권한이 소멸된다는 메시지, 등록한 상품이 없어야 한다는 메시지를 띄우고 확인 완료 버튼을 띄운다
3. 확인 완료 버튼을 다시 입력한다. 	//4탈퇴 완료 메시지를 띄운다.
	  			  

판매 완료된 상품 내역 조회 
1. 판매자센터->판매 완료 의류 상품 조회 버튼을 누른다.	//2. 판매 완료된 상품의 list를 상품명의 오름차순으로 정렬해서 출력한다 

상품 정보 조회(검색) 
1. 회원이 검색 조건으로 상품명, 평균 구매만족도 중에 하나를 선택하여 입력한다	//2.시스템은 검색 조건에 맞춰 해당하는 상품 리스트를 출력한다.

상품 구매
1. 출력된 상품 리스트 중에 구매를 원하는 상품을 선택한다.		//2.선택한 상품의 상세한 정보를 출력한다.
3. 구매 확인 버튼을 누른다.						//4. 상품 구매를 완료한다.
