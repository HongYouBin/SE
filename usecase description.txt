회원 가입
1.사용자가 기본 정보로 이름, 주민번호, 주소, 이메일을 입력하고 ID/Password를 입력한다.  //2-1.기본정보와 ID가 중복이 있는지 확인한다. 중복이 없다면 회원 가입 완료 메시지를 띄운다
									    //2-2. 중복이 발생하면 중복 발생 메시지를 띄우고 다시 입력하게 한다
탈퇴
1.회원이 탈퇴 버튼을 입력한다.	//2.탈퇴와 동시에 시스템 사용 권한이 소멸된다는 메시지, 등록한 상품이 없어야 한다는 메시지를 띄우고 확인 완료 버튼을 띄운다
3. 확인 완료 버튼을 다시 입력한다. 	//4-1 등록한 상품이 있는지 확인한 후 등록한 상품이 없으면 탈퇴하고 탈퇴 완료 메시지를 띄운다.
	  			  4-2 등록한 상품이 남아있으면 등록한 상품이 남아있어 탈퇴를 하지 못한다는 메시지를 띄운다

판매 완료된 상품 내역 조회 
1. 회원이 판매 완료 의류 상품 조회 버튼을 누른다.	//2. 판매 완료된 상품의 list를 상품명의 오름차순으로 정렬해서 출력한다 

상품 정보 조회 및 상품 구매 기능
1. 회원이 검색 조건으로 상품명, 평균 구매만족도 중에 하나를 선택하여 입력한다	//2.시스템은 해당하는 상품 리스트를 출력한다.
3. 출력된 상품 중에 구매를 원하는 상품을 선택한다.				//4.선택한 상품의 상세한 정보를 출력한다.
5.구매 확인 버튼을 누른다.							//6. 상품 구매를 진행, 완료한다.
